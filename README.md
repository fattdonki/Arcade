# Arcade

Arcade is a minimal, Material You-themed game launcher. It allows you to block offline games from accessing the internet using a VPN. This saves battery, blocks ads, and protects your privacy.

## Features
* Distraction free launcher letting you focus on your library, with clean Material design
* Block selected games from accessing the network
* Rename games
* Automatically enable or disable Do Not Disturb for selected games

## Permissions
Arcade requires DND permissions, notification permissions, and VPN permissions.
